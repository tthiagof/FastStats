#  PyDataLens

![Python](https://www.python.org)

Uma ferramenta poderosa e intuitiva de análise de dados que transforma arquivos brutos em dashboards interativos em segundos. 

##  O que o projeto faz?
Este analisador automatiza o processo de Exploração de Dados (EDA). Basta carregar seu arquivo e o PyDataLens cuida da limpeza, estatísticas descritivas e visualização gráfica, eliminando a necessidade de escrever código repetitivo para cada novo dataset.

##  Funcionalidades Principais
- **Upload Fácil:** Suporte para arquivos CSV, Excel e JSON.
- **Análise Estatística:** Resumo automático de médias, correlações e outliers.
- **Visualização Interativa:** Gráficos dinâmicos que permitem zoom e filtros em tempo real.
- **Data Cleaning:** Identificação rápida de valores nulos e duplicados.

## Tecnologias
- **[Python](https://www.python.org)** - Linguagem base.
- **[Pandas](https://pandas.pydata.org)** - Manipulação e tratamento de dados.
- **[Plotly](https://plotly.com)** - Gráficos interativos de alta qualidade.
